나루나루

📅 2025-06

지역 맛집찾기 어플

🛠️ 사용 기술

Flutter Node.js MongoDB ChatGPT API Kakao API

📋 프로젝트 개요

나루나루는 사용자의 위치 기반으로 주변 맛집을 추천해주는 모바일 어플리케이션입니다. ChatGPT API를 활용한 AI 맞춤형 추천 시스템과 카카오맵 API를 통한 직관적인 지도 인터페이스를 제공합니다.

✨ 주요 기능

  • GPS 기반 주변 맛집 검색
  • ChatGPT API를 활용한 개인화된 맛집 추천
  • 카카오맵 API 연동으로 실시간 위치 확인
  • 사용자 리뷰 및 평점 시스템
  • 즐겨찾기 및 방문 기록 관리
  • 음식 카테고리별 필터링

💻 개발 과정

프론트엔드: Flutter를 사용하여 iOS와 Android를 동시에 지원하는 크로스 플랫폼 앱으로 개발했습니다. Provider 패턴을 사용하여 상태 관리를 구현했습니다.

백엔드: Node.js와 Express로 RESTful API 서버를 구축했으며, MongoDB를 사용하여 사용자 데이터와 리뷰 정보를 저장했습니다.

API 연동: ChatGPT API를 통해 사용자의 취향을 분석하고 맞춤형 맛집을 추천하는 기능을 구현했으며, 카카오맵 API로 지도와 위치 서비스를 통합했습니다.

🎯 도전 과제 및 해결

문제: ChatGPT API와 카카오맵 API의 비동기 처리가 복잡하고, 간혹 응답 시간이 길어져 사용자 경험이 저하되는 문제가 있었습니다.

해결: async/await 패턴을 사용하여 비동기 코드를 깔끔하게 정리했고, 로딩 인디케이터를 추가하여 사용자에게 진행 상황을 명확히 표시했습니다. 또한 자주 사용되는 데이터는 캐싱하여 API 호출을 최소화했습니다.

📚 배운 점

외부 API 연동 방법과 데이터베이스 설계에 대해 실무적인 경험을 쌓았습니다. 특히 사용자 경험을 고려한 UI/UX 디자인의 중요성을 깨달았으며, 비동기 프로그래밍과 에러 핸들링의 중요성을 배웠습니다.
