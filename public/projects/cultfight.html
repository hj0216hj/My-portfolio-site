Cult Fight

📅 2024-12

2D 횡스크롤 액션 도트 그래픽 게임

🛠️ 사용 기술

Unity C#

📋 프로젝트 개요

Cult Fight는 2D 횡스크롤 액션 게임으로, 도트 그래픽 스타일로 제작되었습니다. 플레이어는 다양한 적들과 싸우며 스테이지를 진행하고, 각 스테이지마다 등장하는 보스를 물리쳐야 합니다.

✨ 주요 기능

  • 부드러운 캐릭터 애니메이션
  • 다양한 적 AI 패턴
  • 스테이지별 보스전
  • 아이템 및 업그레이드 시스템
  • 도트 그래픽 기반의 레트로 스타일

💻 개발 과정

Unity 엔진을 사용하여 개발했으며, C# 스크립트로 게임 로직을 구현했습니다. 특히 캐릭터의 부드러운 움직임을 위해 애니메이션 시스템을 세밀하게 조정했고, Physics2D를 활용하여 자연스러운 충돌 처리를 구현했습니다.

적 AI는 FSM(Finite State Machine) 패턴을 사용하여 상태 기반으로 설계했으며, 각 적마다 고유한 행동 패턴을 부여했습니다.

🎯 도전 과제 및 해결

문제: 캐릭터와 적의 충돌 처리에서 간헐적으로 관통 현상이 발생했습니다.

해결: Rigidbody2D의 Collision Detection을 Continuous로 변경하고, Physics2D 설정에서 Fixed Timestep을 조정하여 문제를 해결했습니다.

📚 배운 점

게임 개발의 전반적인 과정을 경험할 수 있었고, 특히 캐릭터 애니메이션과 충돌 처리에 대해 깊이 이해하게 되었습니다. 또한 게임의 난이도 조절과 플레이어 경험 설계의 중요성을 배웠습니다.
