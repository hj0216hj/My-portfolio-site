My Trip Planner

2025년 3~6월

여행을 계획하고 관리하는 어플리케이션

사용 기술

Android Studio Java Firebase Realtime Database Firestore

프로젝트 개요

팀명: 김김박
팀장: 박현준 (202144005)
팀원: 김승우 (202144031), 김민수 (202344024)
개발 기간: 2025년 3월 4일 ~ 6월 20일
개발 환경: Android Studio, Java, Firebase

My Trip Planner는 단순한 여행 정보 검색의 개념을 확장하여 체계적으로 여행을 계획하고 관리하고자 하는 사람들을 위해 설계된 여행 계획 관리 안드로이드 애플리케이션입니다. 현대 사회에서 여행에 대한 관심은 지속적으로 증가하고 있으며, 많은 사람들이 개인 여행, 가족 여행, 그룹 여행 등 다양한 형태의 여행을 계획하고 실천하려고 노력하고 있습니다.

프로젝트 목적

코로나19 팬데믹 이후 폭발적으로 증가하는 해외여행 수요에 맞춰, 여행자들이 보다 체계적이고 효율적으로 여행 계획을 수립하고 관리할 수 있는 통합 솔루션을 제공하고자 했습니다. 2021년 대비 2024년 해외여행자 수가 약 30배 이상 증가한 상황에서, 기존의 분산된 여행 정보 관리 방식의 한계를 극복하고 사용자 중심의 편리한 여행 계획 도구를 개발하는 것이 본 프로젝트의 핵심 목적입니다.

특히 20~30대 그룹에서 하나의 트렌드로 여겨지는 개인 맞춤형 여행에 대한 정보 공유 기능 및 실시간 일정 관리 시스템을 구축하여, 사용자가 여행 전 계획 단계부터 여행 중 실시간 관리, 여행 후 기록 보관까지 전 과정을 하나의 애플리케이션에서 seamless하게 처리할 수 있도록 하는 것을 목표로 했습니다.

주요 기능

1. 사용자 인증

  • 회원가입, 로그인, 로그아웃, 비밀번호 재설정
  • 자동 로그인 유지 기능
  • 사용자 프로필 관리

2. 여행 계획 생성

  • 새로운 여행 프로젝트 생성 및 기본 정보 입력
  • 여행 기본 정보 입력
  • 여행 목록 조회 및 관리
  • 여행 계획 수정 및 삭제

3. 마이페이지

  • 이름, 이메일, 성별 정보 수정 가능
  • 현재 비밀번호와 새 비밀번호 입력 후 변경 가능
  • 계정 삭제 시 팝업 확인 메시지 출력 후 버튼 클릭 시 계정 삭제

4. 예산 관리

  • 카테고리별 예산 설정 및 지출 기록
  • 실시간 예산 사용 현황 추적
  • 지출 내역 조회 및 수정

5. 지도 및 길찾기

  • 원하는 위치를 검색, 구글Maps와 연결하여 길찾기 기능
  • 목적지 검색 및 마커 표시
  • GPS를 통한 현재 위치 확인

개발 환경

  • 개발 플랫폼: Android Studio (최신 안정 버전)
  • 프로그래밍 언어: Java (JDK 8 이상)
  • 빌드 시스템: Gradle 빌드 도구
  • 데이터베이스: Firebase Realtime Database 및 Firestore
  • 인증 시스템: Firebase Authentication
  • 타겟 API: Android API Level 21 이상 (Android 5.0 Lollipop)
  • 지원 디바이스: 스마트폰 및 태블릿 (화면 크기 4.0" ~ 10.1")

개발 방법

  • 애자일 개발 방법론: 2주 단위 스프린트 진행
  • 반복적 개발: 기능별 우선순위에 따른 단계적 개발
  • 지속적 통합: Git을 활용한 버전 관리 및 협업

역할 분담

  • 팀장 (박현준): 전체 프로젝트 관리, UI/UX 설계, 최종 통합
  • 팀원 1 (김승우): Firebase 연동, 데이터베이스 설계, 백엔드 로직
  • 팀원 2 (김민수): 프론트엔드 개발, 사용자 인터페이스 구현

도전 과제 및 해결

문제: Firebase의 실시간 동기화 시 여러 사용자가 동시에 같은 데이터를 수정할 때 데이터 충돌이 발생하는 문제가 있었습니다.

해결:

  • Firebase Transaction을 사용하여 동시성 문제를 해결
  • 적절한 데이터 구조를 설계하여 충돌 가능성을 최소화
  • 오프라인 상태에서도 작동하도록 Firebase의 오프라인 지속성 기능을 활용

배운 점

  • 팀 프로젝트로 진행하면서 협업 능력과 Git을 활용한 버전 관리의 중요성
  • Firebase를 활용한 실시간 데이터 동기화와 클라우드 기반 백엔드 개발 경험
  • 안드로이드 앱 개발의 전반적인 프로세스를 이해
