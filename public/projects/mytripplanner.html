My Trip Planner

📅 2025-06

여행을 계획하고 관리하는 어플

🛠️ 사용 기술

Android Studio Java Firebase Realtime Database Firestore

📋 프로젝트 개요

My Trip Planner는 여행 계획을 세우고 일정을 관리할 수 있는 안드로이드 어플리케이션입니다. Firebase를 활용하여 실시간 데이터 동기화를 지원하며, 친구들과 여행 계획을 공유하고 함께 편집할 수 있는 협업 기능을 제공합니다.

✨ 주요 기능

  • 여행 일정 생성 및 관리
  • 날짜별 세부 계획 작성 (시간, 장소, 메모)
  • 예산 관리 및 지출 내역 기록
  • 준비물 체크리스트 기능
  • Firebase를 통한 실시간 동기화
  • 여행 사진 앨범 및 추억 기록
  • 친구 초대 및 공동 편집

💻 개발 과정

Android Studio에서 Java로 개발했으며, MVVM(Model-View-ViewModel) 아키텍처 패턴을 적용하여 코드의 유지보수성을 높였습니다.

Firebase Realtime Database는 실시간 채팅과 일정 공유에 사용했고, Firestore는 구조화된 데이터(여행 정보, 사용자 프로필)를 저장하는 데 활용했습니다.

Material Design 가이드라인을 따라 직관적이고 사용하기 편한 UI를 구현했으며, RecyclerView와 CardView를 활용하여 리스트 형태의 데이터를 효율적으로 표시했습니다.

🎯 도전 과제 및 해결

문제: Firebase의 실시간 동기화 시 여러 사용자가 동시에 같은 데이터를 수정할 때 데이터 충돌이 발생하는 문제가 있었습니다.

해결: Firebase Transaction을 사용하여 동시성 문제를 해결했고, 적절한 데이터 구조를 설계하여 충돌 가능성을 최소화했습니다. 또한 오프라인 상태에서도 작동하도록 Firebase의 오프라인 지속성 기능을 활용했습니다.

📚 배운 점

팀 프로젝트로 진행하면서 협업 능력과 Git을 활용한 버전 관리의 중요성을 배웠습니다. Firebase를 활용한 실시간 데이터 동기화와 클라우드 기반 백엔드 개발 경험을 쌓았으며, 안드로이드 앱 개발의 전반적인 프로세스를 이해하게 되었습니다.
